# Leave Calendar Dashboard
This project leverages Microsoft SharePoint as the central data source for storing employee leave records, and utilizes Microsoft Power Apps to develop a dynamic, calendar-based interface that visually displays team members’ vacation schedules in an intuitive and interactive format.

# Key Features:

SharePoint Data Backend:
A dedicated SharePoint list serves as the database, capturing key leave details including employee name, leave type, start date, end date, and status.
This ensures centralized, secure, and structured data management accessible across the organization.
# Power Apps Frontend – Calendar View:
A custom Power App was developed featuring a monthly calendar layout (as shown in the “Holiday Calendar” interface), allowing managers to view leave schedules at a glance.
Color-coded bars represent different employees or leave types (e.g., green for approved leave, red for pending or special leave), enabling quick visual identification.
The app supports filtering by department (e.g., “Customer Service”) and displays both individual and overlapping leave periods seamlessly.
# Real-Time Visibility for Management:
Managers can simply open the calendar to instantly see which employees are on leave on any given day — eliminating the need for manual Excel checks or back-and-forth communication.
The intuitive UI promotes efficient workforce planning and coverage management.
Outcome:
This solution transforms static, manual leave tracking into a live, visual, and interactive tool — empowering managers with real-time visibility and significantly improving operational efficiency.
